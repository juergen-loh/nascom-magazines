LINK BLOCK E (LKSE) – Nascom I/O decod

This is provided for Nascom users who have not got the Nas-10 signal decoded anywhere else in their system. This provides all the decoding necessary for port FFhex.

All the Link connections made on the beard are done so using pieces of wire pushed into gold plated socket pins. Once you have decided how you ant your system configuring it is an easy matter (though fiddly at times) to cut the wire to appropriate lengths and link the pads. It is purely a matter of pulling out the appropriate Links again in link blocks A, 8 and/or C to reconfigure the memory map, page layout and ‘page-on-reset. As long as link block D connections, the power optront, are not changed, the memory contents remain unchanged.
