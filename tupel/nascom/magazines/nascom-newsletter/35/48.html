Using the Emulator.

Being simple-minded, I tend to use one extremely basic loader routine which transfers a 2K block of RAN from the Nascom to the emulator. The routine is relocatable but resides in ROM at address D42F which happens to be convenient in my systee. The Foutine is so basic that, as it stands, the data to be transferred must reside at aBdress 4000H in the Nascom memory. Tt would be easy enough to pick up an extra parameter with Nas-Sys E command to improve this...
