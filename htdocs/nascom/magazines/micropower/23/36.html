Nasprint Modification.

If you try to use Nasprint with the RAM version of ZEAP, you will find at keeps giving the message “uninitiated”. This is because it tests memory location 0C7? ta check that “DC” has been put there by ZEAP. However, the RAM version puts “1D” there instead when cold started. Also, after checking that ZEAP is started, Nasprint attempts to return control by jumping to DOOS. The call for RAM ZEAP is 1003. Therefore, to make Nasprint work properly with RAM ZEAP we must change the locations BI4F and B17D in Nasprint to 1D and 10 respectively. This can be done using the Modify command if Nasprint is in RAM but an EPROM programmer is required 1f you have an EPROM version.
