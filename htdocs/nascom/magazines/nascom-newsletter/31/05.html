Nascom 1 & 2 Graphics ROM Editor

by Z.A. Lesiakowski

For those of you with an EPROM blower wanting to create your own set of characters, the accompanying program will allow you to easily create, change and amend each character cell. It is written in machine code and will run under Nas-Sys.

As most of you are already aware, each character cell for the NI is made Up of 16 rows of 8 dots whereas for the NZ it is 14 rows of B dots, the Tast two rows being unused by the NZ. As there are 128 characters in a character EPROM, the EPROM must be 16% 128 bytes, ie. 2k. On executing the program you will be asked whether you are programming an Ni or N2 EPROM followed by the origin of the 2k of RAM where the characters will be stored.
