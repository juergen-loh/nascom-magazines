Sarten up those Basic Listings !

by D.F. Gutteridge

The following machine code routine enables Basic prograss to be 1isted with indented FOR-NEXT loops, considerably clarifying program structure.

The example shows that a Basic program may be listed in the normal way, and by leaving Basic via the MONITOR command and

executing the indent routine at @COOH (EC8e), an indented listing is produced clearly showing the program structure.

Notes. 1. The routine is fully relocatable.

2. The routine may be interupted using ESC, and the number of lines displayed may be set by the LINES command as usual.

3. The amount of indenting may be altered by changing the entry at 0CBBH and ECC0H.

The routine only picks up FOR and NEXT at the beginning of Jines. Messy structures like:

170 Aco : FOR J=1 10 10 + PRINT ” 1B NEXT T+ PRINT

Will cause problems. The purpose of the routine is to produce good, visually structured listings to well structured programs,

5. The program is written for Nascom ROM Basic Version 4.7 running under Nas-Sys 1 but it should work with Nas-Sys Sas

wert.
